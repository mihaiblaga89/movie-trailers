### Instructions

1. `git clone <repo>`
2. `cd dir`
3. `yarn`
4. `npm start`

The app should open a new browser in the default browser. If it doesn't, open [http://localhost:3000](http://localhost:3000) manually.

### Running the tests

`npm t`

Pressing `a` will run all the tests.
Pressing `w` will show the help.
